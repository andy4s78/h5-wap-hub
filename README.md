# h5-wap-hub

一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为 HTML 页面资源的存储与展示空间。

## 项目简介

在日常开发与测试中，常需要存放一些独立的 H5 页面、静态演示页、临时工具页或移动端适配页面。  
`h5-wap-hub` 旨在提供一个结构清晰、可方便访问的归档仓库，便于开发者随时查阅、分享或部署这些页面。

## 目录说明

```
h5-wap-hub/
├── pages/          # 存放各独立 HTML 页面，按功能或类型分目录
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库入口页（可选），列出所有已归档页面索引
└── README.md       # 本文件
```

- `pages/`：每个子目录对应一个独立页面，目录名建议使用简短且有意义的英文或拼音。
- `assets/`：若页面有公共资源，可统一放置；也可在每个页面目录内独立管理。
- `index.html`：非必需，但建议添加一个简单的页面列表，方便浏览仓库内容。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开运行。
- 页面应尽量自包含，外部依赖（如 CDN 资源）请注明来源。
- 每个页面目录内可附带 `README.md` 说明该页面的用途、使用方式或注意事项。
- 归档页面不限制主题，只要不违反平台政策即可。

## 维护说明

- 欢迎通过 Pull Request 提交新的页面归档，或对已有页面进行改进。
- 提交前请确保页面无恶意内容、无外部跟踪脚本、无违反开源社区规范的行为。
- 建议为每个页面提供简洁的标题和描述，便于其他开发者理解。
- 本仓库为静态资源集合，不提供后端服务或动态功能。
